# Automated-Test
页面自动化测试框架

测试框架：
python3+Selenium+unittest自动化UI测试框架

（selenium+unittest搭建的WebUI自动化测试框架）

    环境部署： python3、selenium3 
    开发工具： Pycharm
    集成工具： Jenkins
    测试代码托管平台：GitHub
    通过主从服务器执行测试
